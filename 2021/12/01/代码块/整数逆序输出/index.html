整数逆序输出

题目描述

编写函数，将一个整数n逆序输出，例如n = 123456,输出654321

简单实现

代码实现思路:假设x = 1234.

取出x的最后一位4，赋值给ret并输出

x除以10摒弃掉最后一位，因为int型会截断浮点型的小数部分。所以此时x=123

判断条件(x>0)，满足，进入循环

接下来同理，直到x = 1 => 1 / 10 = 0 ,退出循环;

共循环四次，printf分别输出了4 3 2 1，由于数字与数字之间不存在换行和空格，所以输出的结果在一行显示

接下来用递归实现

实现方式都是差不多的 取出个位 => 输出 => 去掉个位 => 判断数字位数 => 大于0 => 继续 | 小于0 => 退出

在这行代码中，print()调用了自身，此时n的值为去掉个位数之后的值，这种在函数体内部调用自身的方式称为递归。
