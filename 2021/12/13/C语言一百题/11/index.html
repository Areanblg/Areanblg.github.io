题目描述

将一个正整数分解质因数。例如：输入90,打印出90=2*3*3*5

相关概念

分解质因数的步骤

60 = 2*30

60 = 2*2*15

60 = 2*3*3*5

思路

若即将分解的数为n,建立2~n的循环,

应该先找到最小的质数k,比如上面的例子，最小的质数k为2

找到质数2时,我们让n = n / 2 接着打印出质数2，此时n = 30；k = 2

再判断30是否能被k整除，若满足=> 打印出k => n = n/2 此时n = 15；若不满足使k+1

n = 15; k = 3;n % k == 0 => n=n/k; =>输出k的值3；此时n = 5；

n = 5 k = 3 不满足 => n%k == 0;所以k+1 = 4=>还是不满足k+1 = 5

此时n = 5 k = 5 n = k 所以打印出k的值然后跳出循环
